## üé¨ VEO3 PROMPT WRITING GUIDE

### Prompt Structure Template:
```
[SCENE STYLE] | [ACTION DESCRIPTION] | [CAMERA MOVEMENT] | [AUDIO LAYER]
```

### 1. SCENE STYLE Elements
Include these visual descriptors:

| Element | Examples |
|---------|----------|
| **Lighting** | golden hour, harsh midday sun, neon-lit, candlelit, moonlit, rim lighting, volumetric fog |
| **Atmosphere** | misty, hazy, clear, stormy, dusty, snowy, rainy |
| **Color Grade** | warm orange tones, cool blue grade, desaturated, high contrast, vintage sepia |
| **Visual Style** | photorealistic, cinematic film grain, anime-style, noir, documentary style |
| **Time of Day** | dawn, morning, midday, golden hour, dusk, night, twilight |

### 2. ACTION DESCRIPTION Elements
Be SPECIFIC about:

| Element | Description |
|---------|-------------|
| **Subject Actions** | Walk, run, turn, reach, grab, push, pull, jump, fall, climb, crawl |
| **Facial Expressions** | Smiles slowly, frowns, eyes widen, tears form, jaw drops, looks shocked |
| **Body Language** | Shoulders tense, relaxes, slouches, stands tall, crosses arms, clenches fists |
| **Movement Speed** | Slowly, quickly, abruptly, gradually, in slow motion, at normal speed |
| **Direction** | Left to right, right to left, toward camera, away from camera, circular |
| **Interaction** | With objects, with other characters, with environment |

### 3. CAMERA MOVEMENT Types

| Movement | Description | Use Case |
|----------|-------------|----------|
| **Static** | Camera doesn't move | Dialogue, dramatic moments |
| **Pan Left/Right** | Horizontal rotation | Reveal, follow action |
| **Tilt Up/Down** | Vertical rotation | Reveal height, show scale |
| **Dolly In/Out** | Move toward/away from subject | Intensify or release tension |
| **Truck Left/Right** | Lateral movement | Follow walking subject |
| **Crane Up/Down** | Vertical movement | Grand reveals, establishing |
| **Orbit** | Circle around subject | Hero shots, 360¬∞ reveal |
| **Handheld** | Slight natural shake | Realism, tension, documentary |
| **Steadicam** | Smooth following | Chase scenes, walk and talk |
| **Zoom In/Out** | Optical zoom | Focus attention, comedic effect |
| **Whip Pan** | Fast pan creating blur | Fast transitions, action |

### Camera Angle Terminology:
- **Eye Level**: Natural, neutral
- **Low Angle**: Subject appears powerful/dominant
- **High Angle**: Subject appears vulnerable/small
- **Dutch Angle**: Tilted, creates unease
- **Bird's Eye**: Directly overhead
- **Worm's Eye**: From ground looking up
- **Over-the-shoulder (OTS)**: Character perspective
- **POV Shot**: First person view

### 4. AUDIO LAYER Components

#### Sound Effects (SFX)
| Category | Examples |
|----------|----------|
| **Foley** | Footsteps on wood, fabric rustling, keys jingling, paper shuffling |
| **Ambient** | Wind howling, rain pattering, crowd murmuring, traffic noise, birds chirping |
| **Impact** | Punches landing, doors slamming, glass breaking, explosions, crashes |
| **Mechanical** | Engine revving, gears clicking, machines whirring, computer beeps |
| **Elemental** | Fire crackling, water splashing, thunder rumbling, ice cracking |
| **Creature** | Growling, roaring, hissing, wings flapping, hooves clopping |

#### Background Music (BGM)
| Element | Description |
|---------|-------------|
| **Genre** | Orchestral, electronic, jazz, rock, ambient, folk, cinematic |
| **Mood** | Tense, uplifting, melancholic, mysterious, heroic, romantic, comedic |
| **Instruments** | Strings swelling, piano melody, percussion building, synth pads |
| **Dynamic** | Soft underscore, building intensity, crescendo, diminuendo |
| **Tempo** | Slow and deliberate, moderate pace, fast-paced, accelerating |

#### Speech/Dialogue
| Element | Format |
|---------|--------|
| **Direct Dialogue** | "What are you doing here?" spoken in whispered, fearful tone |
| **Narration** | Deep male voiceover: "The journey had only begun..." |
| **Vocal Quality** | Raspy, clear, echoing, muffled, trembling, confident |
| **Emotion** | Anger, sadness, excitement, fear, confusion, determination |
| **Volume** | Whispered, normal speaking, shouting, barely audible |

## üéØ PROMPT QUALITY CHECKLIST

For each `veo3_prompt`, verify it includes:

- [ ] **Scene Style**: Lighting, atmosphere, visual quality
- [ ] **Action Breakdown**: What exactly happens, in what order
- [ ] **Subject Directionality**: Which way subjects move, face, look
- [ ] **Speed/Timing**: Slow motion, normal, fast, when things happen
- [ ] **Camera Movement**: Full description of camera behavior
- [ ] **Emotional Beat**: What feeling the shot conveys
